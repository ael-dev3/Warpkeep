## Direction to study

- Full-bleed cinematic fantasy environment with the castle dominating the left side.
- Large, high-contrast title and restrained tagline across the upper field.
- Simple vertical menu on the open right side, separated with fine ornamental rules.
- Warm sunset stone and antique-gold typography contrasted by concentrated violet warp energy.
- Strong depth cue from the foreground gateway to the distant hostile citadel.

Use this as composition, hierarchy, atmosphere, and menu-treatment reference. It is not wired into the application as a runtime asset.
