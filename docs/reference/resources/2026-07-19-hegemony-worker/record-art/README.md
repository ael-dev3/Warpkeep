# Hegemony Worker record art

This directory records the exact transparent Worker illustration supplied for
the Worker inspection surface. The checked-in runtime derivative is
same-origin and digest-pinned; it is decorative UI art, not worker identity or
authority.

- [Manifest](manifest.json)
- Runtime asset: `public/images/realm/hegemony-worker-record.webp`

The source PNG is not retained in the repository. Rebuild the derivative only
from the owner-supplied source whose filename, dimensions, and SHA-256 are
pinned in the manifest, using the Worker record-art installer. Do not fetch a
replacement image from the internet.
