# Hegemony Wheat Farm runtime assets

This record pins the three exact owner-supplied Royal Harvest Windmill GLBs
selected for draft Tier 1 Food-node work. The delivery package and editable
masters are not checked in or fetched by routine builds; only the named,
digest-bearing public runtime files are served after a separately approved
merge and deployment.

The supplied manifest marks its runtime exports as provisional. This review
does not rewrite them or represent that metadata as a production release. It
records the immutable bytes, orientation, bounds, small permitted grounded-rock
embed, visual-only collision guidance, and narrow delivery authority so a
future approved integration can be audited.

The GLB render mesh is not collision, map placement, resource, worker, reward,
or SpacetimeDB authority. See [manifest.json](manifest.json) for exact hashes,
LOD budgets, source-manifest facts, and the authorization boundary.
