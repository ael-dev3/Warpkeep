# Warpkeep Alpha 0.2.0

`0.2.0` is the closed-alpha activation candidate for Warpkeep. It is not a public multiplayer release.

## Included player experience

- Cinematic gateway, Hegemony menu, Credits, and scene-aware title/menu/realm audio.
- Bright Lowlands presentation, a 61-cell Genesis 001 authority layout, and a first keep view.
- Standard Farcaster SIWF on desktop and mobile, with a small remembered-device alpha convenience.
- A deliberately narrow shared-alpha path: Farcaster proof → Warpkeep OIDC → SpacetimeDB admission → one persistent player and castle only after explicit admission.

## Intentionally not included

- Resources, upgrades, units, combat, alliances, chat, seasons, or a public multiplayer loop.
- Any real alpha FID, player, or castle seeded during activation.

## Activation condition

The canonical Pages domain and HTTPS are ready. The release tag is created only after bridge discovery/JWKS, the private direct Maincloud auth-epoch procedure call, non-destructive module publish, 61-cell seed, empty whitelist verification, and remote denial test all pass. Until then, the shared-alpha kill switch remains disabled and the public experience stays menu-safe.
