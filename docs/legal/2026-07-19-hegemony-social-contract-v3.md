# Hegemony Social Contract V2 → V3

The published [Hegemony Social Contract](../../public/social-contract/index.html)
now uses version `2026-07-19-HEGEMONY-SOCIAL-CONTRACT-V3`. It is incorporated
into entry-agreement bundle `2026-07-19-hegemony-entry-agreement-v3`.

V3 makes the fiction boundary explicit, keeps in-game sacrifice and any future
tribute flow forward-looking, protects good-faith dissent, and states that
observable conduct—not beliefs or lawful identity—sets the citizenship standard.
The current Alpha still has no sacrifice system, tribute flow, Ouster faction,
or complete core strategy loop.

The existing private `alpha_terms_acceptance_v1` table remains append-only. V2
rows are retained as historical evidence and are never rewritten or deleted.
The current entry and gameplay gates compare only the V3 bundle, so every
participant—including previously admitted founders—must make a fresh,
unchecked V3 acceptance. A new acceptance adds a V3 row keyed by
`FID:2026-07-19-hegemony-entry-agreement-v3`.

This change does not add sacrifice, token burning, premium-unit redemption,
Ouster participation, IP-ban tooling, or any other gameplay described as future
in the Charter.
