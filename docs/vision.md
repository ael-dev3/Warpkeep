# Warpkeep vision

Warpkeep is a Farcaster-native asynchronous strategy world where every FID can have a castle.

The world is live. The first Hegemony keep stands in a deterministic Lowlands realm, standard website SIWF establishes identity through a trusted OIDC bridge, and SpacetimeDB already owns the admission and shared-world authority. The admission list is deliberately closed while the strategy systems grow one verified slice at a time.

The intended experience is:

1. Enter a premium fantasy world through the WARPKEEP gateway.
2. Verify a Farcaster identity without making the browser authoritative.
3. Return to one persistent keep and a shared frontier that changes over time.
4. Build, train, scout, raid, ally, and rule through deterministic server-owned systems.
5. See history, social relationships, court reports, and world events make that state feel alive.

Warpkeep is not a dashboard, financial product, chain-execution client, or local campaign slot. Farcaster identity and social context support shared play; SpacetimeDB reducers remain the sole authority for ownership, resources, timers, movement, and combat.

AI may write lore, reports, summaries, quests, and recommendations from read-only snapshots. It must never decide authoritative game outcomes.
